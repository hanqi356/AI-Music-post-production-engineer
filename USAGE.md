# AI音乐后期工程师 - 使用说明

## 软件简介

AI音乐后期工程师是一款专业的音乐修音软件，集成了音准调校、EQ调节、录音工程和母带制作四大核心功能，适用于音乐制作人、音频工程师和音乐爱好者。

## 功能介绍

### 1. 音准调校
- **音准调整**：支持±12半音的精确音高调整
- **实时预览**：调整参数时可实时听到效果
- **智能修正**：基于音频分析的智能音准修正

### 2. EQ调节
- **多频段均衡**：9个专业频段的均衡控制
- **可视化频谱**：实时显示音频频谱分布
- **预设配置**：多种常用均衡预设

### 3. 录音工程
- **多轨录音**：支持多轨道同时录音
- **轨道管理**：灵活的轨道编辑和管理
- **混音功能**：专业级音频混音处理
- **项目导出**：支持多种格式导出

### 4. 母带制作
- **压缩器**：参数可调的动态压缩处理
- **限制器**：峰值限制保护
- **立体声增强**：空间感和宽度调节
- **预设模式**：平衡、响亮、动态三种模式

## 安装和启动

### 系统要求
- Windows 7 或更高版本
- Python 3.8 或更高版本
- 至少 4GB RAM

### 安装步骤
1. 确保已安装 Python 3.8+
2. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```
3. 启动软件：
   ```
   python main.py
   ```

### 快捷启动
- Windows 用户可运行 `launch.bat` 脚本
- 或运行 `python run_app.py`

## 使用指南

### 基本工作流程
1. **加载音频**：点击菜单"文件" → "加载音频"导入音频文件
2. **音准调校**：在"音准调校"标签页调整音高参数
3. **EQ调节**：在"EQ调节"标签页进行频率均衡
4. **录音工程**：在"录音工程"标签页进行多轨编辑
5. **母带制作**：在"母带制作"标签页进行最终处理
6. **保存音频**：点击菜单"文件" → "保存音频"导出处理后的音频

### 音准调校详细操作
- 使用滑块调整音准偏移（单位：半音）
- 数值为正表示升高音高，负值表示降低音高
- 点击"应用音准调整"按钮应用效果

### EQ调节详细操作
- 各频段滑块可独立调节（-12dB 到 +12dB）
- 实时观察频谱变化
- 点击"应用EQ调节"按钮应用效果

### 录音工程详细操作
- 点击"开始录音"按钮进行录音
- 点击"添加音轨"增加新轨道
- 使用"混音"按钮合并所有轨道
- 点击"导出项目"保存整个项目

### 母带制作详细操作
- 调整压缩器阈值（-60dB 到 0dB）
- 调整限制器上限（-20dB 到 0dB）
- 调整立体声宽度（0% 到 200%）
- 选择预设模式（平衡、响亮、动态）
- 点击"应用母带处理"按钮应用效果

## 注意事项

- 大文件处理可能需要较长时间，请耐心等待
- 录音功能需要麦克风设备支持
- 建议使用高品质音频文件获得最佳效果
- 处理前请备份原始音频文件

## 技术支持

如遇到技术问题，请检查：
1. 是否正确安装了所有依赖包
2. Python 版本是否符合要求
3. 音频文件格式是否支持（WAV、MP3、FLAC、M4A）
