# AI音乐后期工程师 - 更新日志

## v1.1.0 (2026年2月2日) - 对比功能与pyworld支持

### 新增功能
- **音频对比功能**：新增处理前后音频波形和频谱对比显示
  - 支持波形对比、频谱对比和叠加显示模式
  - 每个标签页独立的对比控制
  - 全局对比开关
  - 一键重置功能

- **pyworld库支持**：为高级音准校正提供更好的支持
  - 添加pyworld作为可选依赖
  - 创建详细的安装指导文档
  - 提供多种安装方案（conda、预编译wheel、VS Build Tools）
  - 实现优雅的降级处理机制

### 功能增强
- **界面改进**：
  - 在音准调校、EQ调节、母带制作标签页添加对比控制
  - 新增"视图"和"工具"菜单
  - 优化可视化组件，支持多种显示模式

- **代码优化**：
  - 扩展MatplotlibWidget类，新增plot_comparison和plot_overlay方法
  - 实现音频数据备份机制
  - 优化内存使用（限制采样点数量）

- **用户体验**：
  - 处理完成后自动显示对比效果
  - 支持随时切换对比/单独显示模式
  - 与现有UI风格保持一致

## v1.0.0 (初始版本) - 基础功能

### 核心功能
- **音准调校**：自动检测并修正音高偏差
- **EQ调节**：多频段均衡器，精细频率控制
- **录音工程**：多轨录音和编辑
- **母带制作**：专业级音频母带处理
- **反AI痕迹处理**：智能保持人声自然特征
- **跨平台支持**：Windows、macOS、Linux支持

### 智能功能
- 智能一键音准校准
- 智能一键EQ调节
- 智能一键母带处理
- 自动参数推荐

### 技术特性
- 使用PyQt5构建跨平台GUI
- 集成librosa、scipy、numpy等音频处理库
- 实现实时音频可视化
- 支持多种音频格式
